Boredom

Challenge too low relative to skill. Mind wanders, disengagement occurs.

Flow

Optimal challenge-skill balance. Deep focus, time distortion, effortless performance.

Anxiety

Challenge exceeds skill level. Stress and overwhelm prevent optimal performance.

Psychology concept: Flow State (Csikszentmihalyi) occurs when challenge and skill are optimally matched. This creates a state of effortless concentration, time distortion, and peak performance.
